Klavi — Support

Real mechanical, typewriter, and acoustic keyboard sounds that play as you type, on your Mac.

Getting started

On first launch, Klavi asks for the Input Monitoring permission. This lets the app detect when a key is pressed so it can play a sound. Klavi only detects that a key was pressed — never what you type.

If you skipped the prompt, grant it manually: System Settings → Privacy & Security → Input Monitoring, then enable Klavi and relaunch the app.

Frequently asked questions

I don't hear any sound. Make sure the "Sound on" toggle is enabled, the volume slider is up, and that Klavi has Input Monitoring permission (see above).

Can I change the keyboard sound? Yes — open the sound library and pick any profile (IBM Model M, Cherry MX, typewriters, acoustic instruments, and more). Use the Preview button to audition before selecting.

Does Klavi record what I type? No. Klavi never reads or stores your keystrokes. See our Privacy Policy.

Can Klavi start automatically? Yes — enable "Launch Klavi at login" in the Advanced settings.
